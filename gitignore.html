Global gitignore

Mar 27, 2016

My view is that you should avoid polluting .gitignore files in your project with a bunch of junk that’s unrelated to the project itself. For example, files generated by your IDE or the operating system. A good way to achieve this is by setting up a global gitignore file.

This file needs to be created at $XDG_CONFIG_HOME/git/ignore or $HOME/.config/git/ignore. Its structure should be like any other gitignore file.

Here’s an example of my own global gitignore file:

# macOS
*~
.DS_Store
.AppleDouble
.LSOverride

# IDEs
.idea
.vscode
*.iml

# Thumbnails
._*

# Jupyter Notebooks
.ipynb_checkpoints

Additional documentation: https://git-scm.com/docs/gitignore
